# Russian UI Style

- Use natural Russian UI phrasing, not English word order.
- Prefer concise infinitive command labels: `Открыть папку`, `Сохранить версию`, `Удалить задачу`.
- Use correct case endings for nouns, especially after verbs and prepositions.
- Placeholders can make case endings fragile. Rephrase around placeholders when the required case is unknown.
- Keep `Pinokio`, `Git`, `GitHub`, `HTTP`, `HTTPS`, code identifiers, paths, and URLs literal.
- Preserve placeholders exactly: `{name}` stays `{name}`.
- Use Russian punctuation naturally in full questions and warnings.
- Keep buttons short and direct.
- Keep destructive actions clear; do not confuse reset with refresh.
- Use consistent terms for app, plugin, launcher, task, workspace, checkpoint, and terminal.
