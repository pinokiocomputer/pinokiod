# Spanish UI Style

- Use natural Spanish UI phrasing, not English word order.
- Prefer infinitive command labels: `Abrir carpeta`, `Guardar versión`, `Restablecer selección`.
- Use sentence-style capitalization for labels.
- Use articles and prepositions when Spanish requires them; avoid literal English noun chains.
- Use inverted punctuation for full questions: `¿... ?`.
- Keep `Pinokio`, `Git`, `GitHub`, `HTTP`, `HTTPS`, code identifiers, paths, and URLs literal.
- Preserve placeholders exactly: `{name}` stays `{name}`.
- Rephrase around placeholders when gender or number would otherwise be wrong.
- Keep command labels short; put explanations in helper text.
- Use consistent terms for app, plugin, launcher, task, workspace, and checkpoint.
