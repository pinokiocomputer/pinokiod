# Traditional Chinese UI Style

- Use Traditional Chinese terminology.
- Use natural concise UI wording, not English word order.
- Prefer short verb-object commands: `й–Ӣе•ҹиіҮж–ҷеӨҫ`, `е„ІеӯҳзүҲжң¬`, `еҲӘйҷӨд»»еӢҷ`.
- Do not insert spaces between Chinese words. Spaces are acceptable around Latin code terms, paths, URLs, and product names when needed.
- Keep `Pinokio`, `Git`, `GitHub`, `HTTP`, `HTTPS`, code identifiers, paths, and URLs literal.
- Preserve placeholders exactly: `{name}` stays `{name}`.
- Rephrase around placeholders when the sentence would otherwise sound like translated English.
- Keep button labels short and direct.
- Keep destructive actions clear; do not confuse reset with refresh.
- Use consistent terms for app, plugin, launcher, task, workspace, checkpoint, and terminal.
