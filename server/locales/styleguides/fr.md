# French UI Style

- Use natural French app UI language, not English word order.
- Prefer infinitive command labels: `Ouvrir le dossier`, `Enregistrer la version`.
- Use sentence-style capitalization for UI labels unless a proper noun requires uppercase.
- Use articles and prepositions naturally; do not translate English noun chains literally.
- Keep `Pinokio`, `Git`, `GitHub`, `HTTP`, `HTTPS`, code identifiers, paths, and URLs literal.
- Use French punctuation naturally in full sentences and questions.
- Preserve placeholders exactly: `{name}` stays `{name}`.
- Rephrase around placeholders when gender or number would otherwise be wrong.
- Keep buttons short and action-oriented.
- Use terms from `terms.json` consistently across settings, app pages, terminal, logs, tasks, and plugins.
