# Brazilian Portuguese UI Style

- Use Brazilian Portuguese, not European Portuguese.
- Use natural app UI phrasing, not English word order.
- Prefer infinitive command labels: `Abrir pasta`, `Salvar versão`, `Restaurar seleção`.
- Use sentence-style capitalization for labels.
- Use articles and prepositions naturally; avoid literal English noun chains.
- Keep `Pinokio`, `Git`, `GitHub`, `HTTP`, `HTTPS`, code identifiers, paths, and URLs literal.
- Preserve placeholders exactly: `{name}` stays `{name}`.
- Rephrase around placeholders when gender or number would otherwise be wrong.
- Keep buttons short and direct.
- Use consistent terms for app, plugin, launcher, task, workspace, and checkpoint.
